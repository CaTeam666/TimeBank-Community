# 爱心超市接口文档

## 1. 获取商品列表

**接口描述**: 获取爱心超市的商品列表，支持按名称搜索和按分类筛选

**请求方法**: `GET`
**请求路径**: `/api/mall/products`

**请求参数 (Query)**:

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| keyword | string | 否 | 搜索关键词 (商品名称) | 米 |
| category | string | 否 | 商品分类 (粮油副食, 日用百货, 医疗健康, 虚拟券卡) | 粮油副食 |

**响应参数 (JSON)**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码 (200成功) |
| message | string | 响应信息 |
| data | array | 商品列表 |
| data[].id | number | 商品ID |
| data[].name | string | 商品名称 |
| data[].image | string | 商品图片URL |
| data[].price | number | 商品价格 (积分) |
| data[].category | string | 商品分类 |
| data[].stock | number | 库存数量 |
| data[].sales | number | 月销量 |

**响应示例**:
```json
{
  "code": 200,
  "message": "查询成功",
  "data": [
    {
      "id": 1,
      "name": "东北大米 5kg",
      "image": "http://example.com/rice.jpg",
      "price": 50,
      "category": "粮油副食",
      "stock": 100,
      "sales": 500
    },
    {
      "id": 2,
      "name": "鲁花花生油 5L",
      "image": "http://example.com/oil.jpg",
      "price": 120,
      "category": "粮油副食",
      "stock": 50,
      "sales": 200
    }
  ]
}
```

**数据库操作**:
- 查询 `tb_product` 表
- 条件: `status = 1` (上架状态)
- 动态条件: 
  - if `keyword`: `name LIKE %keyword%`
  - if `category`: `category = category`

## 2. 获取商品详情

**接口描述**: 获取指定商品的详细信息

**请求方法**: `GET`
**请求路径**: `/api/mall/product/detail`

**请求参数 (Query)**:

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| id | number | 是 | 商品ID | 1 |

**响应参数 (JSON)**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码 (200成功) |
| message | string | 响应信息 |
| data | object | 商品详情 |
| data.id | number | 商品ID |
| data.name | string | 商品名称 |
| data.image | string | 商品图片URL |

| data.category | string | 商品分类 |
| data.stock | number | 库存数量 |
| data.sales | number | 月销量 |
| data.description | string | 商品描述 |

**响应示例**:
```json
{
  "code": 200,
  "message": "查询成功",
  "data": {
    "id": 1,
    "name": "东北大米 5kg",
    "image": "http://example.com/rice.jpg",
    "price": 50,
    "category": "粮油副食",
    "stock": 100,
    "sales": 500,
    "description": "优质东北大米，颗粒饱满..."

```

**数据库操作**:
- 查询 `tb_product` 表
- 条件: `id = #{id}`

## 3. 积分兑换 (创建订单)

**接口描述**: 用户使用积分兑换商品，生成兑换订单

**请求方法**: `POST`
**请求路径**: `/api/mall/exchange`

**请求参数 (JSON)**:

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| userId | string | 是 | 用户ID | 1001 |
| productId | number | 是 | 商品ID | 1 |
| quantity | number | 是 | 兑换数量 | 1 |

**响应参数 (JSON)**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码 (200成功) |
| message | string | 响应信息 |
| data | object | 订单信息 |
| data.orderNo | string | 订单编号 |
| data.verifyCode | string | 核销码 |

**响应示例**:
```json
{
  "code": 200,
  "message": "兑换成功",
  "data": {
    "orderNo": "ORD20250101123456",
    "verifyCode": "882910"
  }
}
```

**数据库操作 (事务)**:
1. **扣减库存**: 更新 `tb_product`，`stock = stock - quantity`，`sales_count = sales_count + quantity` (需检查库存 sufficiency)
2. **扣减积分**: 更新 `sys_user`，`balance = balance - total_price` (需检查余额 sufficiency)
3. **创建订单**: 插入 `tb_exchange_order` (status=0 待核销)
4. **记录流水**: 插入 `tb_coin_log` (type=3 兑换支出, amount=-total_price)

## 4. 获取我的兑换订单列表

**接口描述**: 获取指定用户的兑换订单记录

**请求方法**: `GET`
**请求路径**: `/api/mall/orders`

**请求参数 (Query)**:

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| userId | string | 是 | 用户ID | 1001 |

**响应参数 (JSON)**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码 (200成功) |
| message | string | 响应信息 |
| data | array | 订单列表 |
| data[].id | string | 订单编号/ID |
| data[].userId | string | 用户ID |
| data[].productId | number | 商品ID |
| data[].productName | string | 商品名称 |
| data[].productImage | string | 商品图片 |
| data[].price | number | 商品单价 |
| data[].quantity | number | 兑换数量 |
| data[].totalPrice | number | 总价 |
| data[].status | number | 状态 (0:待核销 1:已核销 2:已取消) |
| data[].verifyCode | string | 核销码 |
| data[].createTime | string | 创建时间 |

**响应示例**:
```json
{
  "code": 200,
  "message": "查询成功",
  "data": [
    {
      "id": "ORD20250101123456",
      "userId": "1001",
      "productId": 1,
      "productName": "东北大米 5kg",
      "productImage": "http://example.com/rice.jpg",
      "price": 50,
      "quantity": 1,
      "totalPrice": 50,
      "status": 0,
      "verifyCode": "882910",
      "createTime": "2025-01-01 12:00:00"
    }
  ]
}
```

**数据库操作**:
- 查询 `tb_exchange_order` 表联查 `tb_product` (获取商品信息)
- 条件: `user_id = #{userId}`
- 排序: `create_time DESC`
