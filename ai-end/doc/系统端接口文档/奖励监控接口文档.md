# 奖励监控接口文档

## 1. 接口概述

Base URL: `/api`

本模块负责查询每月自动发放的奖励记录及异常重试。

## 2. 数据结构

### RankingLog (排名日志)

```typescript
interface RankingLog {
  id: string;
  period: string;        // 期数 (e.g., "2024-11")
  rank: number;          // 排名
  volunteerId: string;
  volunteerName: string;
  volunteerAvatar: string;
  orderCount: number;    // 接单数 (原服务时长)
  rewardAmount: number;  // 奖励金额
  distributionTime: string; // 发放时间
  status: 'SUCCESS' | 'FAILURE';
}
```

## 3. 接口详情

### 3.1 获取奖励发放日志

- **URL**: `/ranking/logs`
- **Method**: `GET`
- **Query Params**:
  - `period`: 期数筛选 (可选)
  - `page`: 页码
  - `pageSize`: 每页数量

#### Response
```json
{
  "code": 200,
  "data": {
      "list": [
          {
              "id": "1",
              "period": "2024-11",
              "rank": 1,
              "volunteerName": "李四",
              "orderCount": 45,
              "rewardAmount": 100,
              "status": "SUCCESS",
              "distributionTime": "2024-12-01 00:00:05"
          }
      ],
      "total": 5
  },
  "message": "success"
}
```

### 3.2 手动触发补发

- **URL**: `/ranking/retry`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "id": "1" // 日志ID
}
```

#### Response
```json
{
  "code": 200,
  "message": "补发指令已提交"
}
```
