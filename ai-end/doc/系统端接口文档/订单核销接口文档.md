# 订单核销接口文档

## 1. 接口概述

Base URL: `/api`

本模块负责商品兑换订单的查询、核销与取消。

## 2. 数据结构

### ExchangeOrder (订单)

```typescript
interface ExchangeOrder {
  id: string;
  orderNo: string;       // 订单号
  volunteerId: string;
  volunteerName: string;
  productId: string;
  productName: string;
  productImage: string;
  cost: number;          // 消耗时间币
  createTime: string;
  verifyCode: string;    // 核销码
  status: 'PENDING' | 'COMPLETED' | 'CANCELLED';
}
```

## 3. 接口详情

### 3.1 获取订单列表

- **URL**: `/order/list`
- **Method**: `GET`
- **Query Params**:
  - `status`: 状态筛选 (ALL, PENDING, COMPLETED, CANCELLED)
  - `keyword`: 搜索关键字 (订单号/志愿者名)
  - `page`: 页码
  - `pageSize`: 每页数量

#### Response
```json
{
  "code": 200,
  "data": {
      "list": [
          {
              "id": "1",
              "orderNo": "ORD-20241224001",
              "volunteerName": "张三",
              "productName": "米油套餐",
              "cost": 500,
              "verifyCode": "123456",
              "status": "PENDING",
              "createTime": "2024-12-24 10:00:00"
              // ... other fields
          }
      ],
      "total": 100
  },
  "message": "success"
}
```

### 3.2 订单核销

- **URL**: `/order/verify`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "id": "1" 
  // 或者使用 "verifyCode": "123456"
}
```

#### Response
```json
{
  "code": 200,
  "message": "核销成功"
}
```

### 3.3 取消订单

- **URL**: `/order/cancel`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "id": "1"
}
```

#### Response
```json
{
  "code": 200,
  "message": "订单已取消"
}
```
