# API 接口文档

## 认证模块

### 用户登录

**接口地址**: `/auth/login` (目前由 `services/auth.ts` 模拟)
**请求方式**: `POST` (虽然是前端模拟，但逻辑上等同)

**请求参数** (Request Body):

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| username | string | 是 | 用户名 (如: admin) |
| password | string | 是 | 密码 (如: 123456) |

**响应参数** (Response Body):

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | number | 状态码 (200表示成功) |
| msg | string | 提示信息 |
| data | object | 数据载体 |
| data.token | string | 身份令牌 |
| data.user | object | 用户信息对象 |

**错误响应**:

- 401 Unauthorized: 账号或密码错误
- 500 Internal Server Error: 系统错误

**示例**:

```typescript
// 请求
const response = await authService.login('admin', '123456');

// 后端原始响应
{
  "code": 200,
  "msg": "success",
  "data": {
    "token": "real-jwt-token-xyz...",
    "user": {
        "id": "ADMIN001",
         // ...
    }
  }
}
```
