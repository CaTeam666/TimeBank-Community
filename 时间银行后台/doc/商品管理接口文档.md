# 商品管理接口文档

## 1. 接口概述

Base URL: `/api`

本模块负责商品（时间币兑换物品）的增删改查及上下架管理。

## 2. 数据结构

### Product (商品)

```typescript
interface Product {
  id: string;          // 商品ID
  name: string;        // 商品名称
  description: string; // 商品描述
  image: string;       // 图片URL
  price: number;       // 兑换所需时间币
  stock: number;       // 库存数量
  status: 'ON_SHELF' | 'OFF_SHELF'; // 状态
  salesCount: number;  // 销量
  category: '粮油副食' | '日用百货' | '医疗健康' | '虚拟券卡'; // 商品分类
}
```

## 3. 接口详情

### 3.1 获取商品列表

- **URL**: `/product/list`
- **Method**: `GET`
- **Description**: 获取所有商品列表（一般后台管理查看所有，含下架的；前台只看上架的。这里默认后台接口）

#### Response (Success)
```json
{
  "code": 200,
  "data": [
    {
      "id": "1",
      "name": "米油套餐",
      "description": "五常大米与食用油组合",
      "image": "https://example.com/img.jpg",
      "price": 500,
      "stock": 100,
      "status": "ON_SHELF",
      "price": 500,
      "stock": 100,
      "status": "ON_SHELF",
      "salesCount": 10,
      "category": "粮油副食"
    }
  ],
  "message": "success"
}
```

### 3.2 获取商品详情

- **URL**: `/product/:id`
- **Method**: `GET`

#### Response (Success)
```json
{
  "code": 200,
  "data": {
    "id": "1",
    "name": "米油套餐",
    // ... other fields
  },
  "message": "success"
}
```

### 3.3 新增商品

- **URL**: `/product/create`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "name": "洗洁精",
  "description": "强力去油",
  "image": "https://...",
  "price": 50,
  "stock": 200,
  "price": 50,
  "stock": 200,
  "status": "ON_SHELF", // Optional, default ON_SHELF
  "category": "日用百货"
}
```

#### Response (Success)
```json
{
  "code": 200,
  "data": {
      "id": "101",
      // ... created object
  },
  "message": "success"
}
```

### 3.4 更新商品

- **URL**: `/product/update`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "id": "1",
  "name": "新名称", // Optional fields to update
  "id": "1",
  "name": "新名称", // Optional fields to update
  "stock": 150,
  "category": "日用百货"
}
```

#### Response (Success)
```json
{
  "code": 200,
  "message": "success"
}
```

### 3.5 更新商品状态 (上下架)

- **URL**: `/product/status`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "id": "1",
  "status": "OFF_SHELF" // or "ON_SHELF"
}
```

#### Response (Success)
```json
{
  "code": 200,
  "message": "success"
}
```

### 3.6 删除商品

- **URL**: `/product/delete`
- **Method**: `POST`
- **Content-Type**: `application/json`

#### Request Body
```json
{
  "id": "1"
}
```

#### Response (Success)
```json
{
  "code": 200,
  "message": "success"
}
```
