# 排行榜接口文档

## 1. 接口概述

Base URL: `/api/client`

本模块提供志愿者接单排行榜相关功能，包括实时排行、历史月度排行及个人接单统计。

## 2. 数据结构

### RankingItem (排行项)

```typescript
interface RankingItem {
  rank: number;              // 排名
  volunteerId: string;       // 志愿者ID
  volunteerName: string;     // 志愿者姓名
  volunteerAvatar: string;   // 志愿者头像
  orderCount: number;        // 接单数
}
```

### MonthlyRanking (月度排行)

```typescript
interface MonthlyRanking {
  period: string;            // 期数 (e.g., "2026-03")
  list: RankingItem[];       // 排行列表
  rewardInfo?: {             // 奖励信息（仅已发放月份有）
    rank: number;
    rewardAmount: number;
  }[];
}
```

### UserOrderStats (用户接单统计)

```typescript
interface UserOrderStats {
  userId: string;            // 用户ID
  totalOrderCount: number;   // 总接单数
  currentMonthCount: number; // 当月接单数
  currentRank: number | null; // 当前排名（无排名则为null）
}
```

### 3.2 根据月份查看历史排行

查询指定月份的排行榜数据（优先从缓存获取，已发放奖励的月份会显示奖励信息）。

- **URL**: `/ranking/monthly`
- **Method**: `GET`
- **需要登录**: 是
- **Query Params**:
  - `period`: 期数 (必填，格式: YYYY-MM，如 2026-02)

#### Response
```json
{
  "code": 200,
  "data": {
    "period": "2026-02",
    "list": [
      {
        "rank": 1,
        "volunteerId": "1",
        "volunteerName": "宋幼之",
        "volunteerAvatar": "/uploads/avatar/1.png",
        "orderCount": 10
      },
      {
        "rank": 2,
        "volunteerId": "3",
        "volunteerName": "张小明",
        "volunteerAvatar": null,
        "orderCount": 7
      }
    ],
    "rewardInfo": [
      { "rank": 1, "rewardAmount": 1000 },
      { "rank": 2, "rewardAmount": 500 },
      { "rank": 3, "rewardAmount": 300 },
      { "rank": 4, "rewardAmount": 300 },
      { "rank": 5, "rewardAmount": 300 }
    ]
  },
  "message": "success"
}
```

#### 错误码
| code | message |
|------|---------|
| 400 | 期数格式错误 |
| 404 | 该月份暂无排行数据 |

### 3.3 查询当前用户接单数

获取当前登录用户的接单统计信息，包括总接单数、当月接单数及当前排名。

- **URL**: `/ranking/my-stats`
- **Method**: `GET`
- **需要登录**: 是

#### Response
```json
{
  "code": 200,
  "data": {
    "userId": "1",
    "totalOrderCount": 45,
    "currentMonthCount": 8,
    "currentRank": 1
  },
  "message": "success"
}
```

#### 说明
- `currentRank`: 当前月份在所有志愿者中的排名，如果不在前100名则返回 `null`
- `totalOrderCount`: 累计所有已完成任务的接单数
- `currentMonthCount`: 当前月份已完成任务的接单数

## 4. 奖励规则

每月1日系统自动统计上月接单数前5名志愿者，发放时间币奖励：

| 排名 | 奖励时间币 |
|------|------------|
| 第1名 | 1000 |
| 第2名 | 500 |
| 第3名 | 300 |
| 第4名 | 300 |
| 第5名 | 300 |
