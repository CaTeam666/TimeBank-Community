**文档名称**: 实名审核接口文档  
**创建时间**: 2024-12-24  
**版本**: v1.0

---

## 1. 获取实名审核列表

**接口地址**: `/api/identity-audits`  
**请求方式**: `GET`  
**描述**: 分页获取实名认证审核任务列表，支持按状态筛选。

### 请求参数

| 参数名 | 类型 | 必选 | 说明 |
|--------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| pageSize | int | 否 | 每页数量，默认 10 |
| status | string | 否 | 审核状态 (PENDING, APPROVED, REJECTED) |

### 响应结果

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": "1",
        "userId": "1001",
        "userName": "张三",
        "submitTime": "2024-05-20 10:00:00",
        "ocrAge": 68,
        "idCardFront": "http://example.com/front.jpg",
        "idCardBack": "http://example.com/back.jpg",
        "ocrName": "张三",
        "ocrIdNumber": "110101195601011234",
        "status": "PENDING",
        "role": "老人"
      }
    ],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}
```

## 2. 获取审核详情

**接口地址**: `/api/identity-audits/:id`  
**请求方式**: `GET`  
**描述**: 获取单个审核任务的详细信息。

### 请求参数

| 参数名 | 类型 | 必选 | 说明 |
|--------|------|------|------|
| id | string | 是 | 审核任务ID |

### 响应结果

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": "1",
    "userId": "1001",
    "userName": "张三",
    "submitTime": "2024-05-20 10:00:00",
    "ocrAge": 68,

    "ocrIdNumber": "110101195601011234",
    "status": "PENDING",
    "role": "老人"
  }
}
```

## 3. 提交审核结果

**接口地址**: `/api/identity-audits/:id/audit`  
**请求方式**: `POST`  
**描述**: 提交对某个实名认证请求的审核结果（通过或驳回）。

### 请求参数

| 参数名 | 类型 | 必选 | 说明 |
|--------|------|------|------|
| status | string | 是 | 审核结果状态 (APPROVED, REJECTED) |
| rejectReason | string | 否 | 驳回原因 (当 status 为 REJECTED 时必填) |

### 请求示例

```json
{
  "status": "REJECTED",
  "rejectReason": "身份证照片模糊"
}
```

### 响应结果

```json
{
  "code": 200,
  "message": "审核成功",
  "data": null
}
```
