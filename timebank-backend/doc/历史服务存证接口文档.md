# 历史服务存证接口文档

## 1. 概述
历史服务存证模块用于展示志愿者完成任务时提交的凭证（如照片），供管理员进行归档查看和审计。

## 2. 数据库映射与分析

### 核心表: `tb_task_evidence` (服务存证表 - **新增**)

| 前端字段 (Evidence) | 数据库字段 (tb_task_evidence) | 说明 |
|---|---|---|
| id | id | 凭证ID |
| taskId | task_id | 关联任务ID |
| volunteerId | volunteer_id | 志愿者ID |
| imageUrl | image_url | 凭证图片URL |
| createTime | create_time | 上传时间 |
| volunteerName | - | *关联查询 `sys_user`* |
| taskTitle | - | *关联查询 `tb_task`* |

## 3. 接口定义

### 3.1 获取存证列表

**URL**: `/evidence/list`
**Method**: `GET`
**Description**: 分页获取服务存证列表，支持按照关键词（任务ID/志愿者信息）搜索。

**请求参数 (Query Params)**:

| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| page | number | 否 | 页码，默认 1 |
| pageSize | number | 否 | 每页条数，默认 12 |
| keyword | string | 否 | 搜索关键词 (任务ID / 志愿者姓名 / 志愿者ID) |

**响应数据**:

```typescript
{
  "code": 200,
  "msg": "success",
  "data": {
    "total": 100,
    "list": [
      {
        "id": "1",
        "taskId": "1024",
        "taskTitle": "社区清洁服务",
        "volunteerId": "V001",
        "volunteerName": "李华",
        "imageUrl": "http://example.com/evidence1.jpg",
        "createTime": "2024-05-20 14:30:00"
      }
    ]
  }
}
```
